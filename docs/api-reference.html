API Reference

Complete API documentation for CollectionFC library.

Overview

CollectionFC provides a comprehensive API for collection forecasting. The library is organized into several modules, each handling specific aspects of the forecasting process.

Module Structure

  • collection_fc.cli - Command-line interface
  • collection_fc.simple - Simple forecasting mode
  • collection_fc.bucket - State bucket definitions
  • collection_fc.flexible - Advanced forecasting options

CLI Interface

The command-line interface provides easy access to CollectionFC functionality.

Main Entry Point

python -m collection_fc [options]

Command Line Arguments

Argument Type Required Description
--mode str Yes Forecasting mode ('simple' or 'flexible')
--input str Yes Path to input CSV file
--forecast_path str No Output path for forecast (default: forecast.csv)
--matrix_path str No Output path for transition matrix (default: transition_matrix.csv)
--validation_path str No Output path for validation (default: validation.json)

Data Structures

Input DataFrame Schema

{ 'account_id': str, # Unique account identifier 'as_of_date': datetime, # Month-end snapshot date 'dpd': int, # Days past due 'balance': float, # Outstanding principal 'recovery_amt': float # Cash collected during month }

Transition Matrix

7×7 numpy array where P[i,j] is the probability of moving from state i to state j.

Cash Collection Vector

7-element numpy array where c[i] is the average cash collected from state i.

Validation Results

{ 'one_step_state_accuracy': float, # State prediction accuracy 'cash_wmape_12m': float, # Weighted MAPE on cash 'cumulative_bias': float # Systematic bias measure }

Exceptions

CollectionFCError

Base exception class for all CollectionFC errors.

DataValidationError

Raised when input data fails validation checks.

InsufficientDataError

Raised when there's insufficient historical data for modeling.

StateAssignmentError

Raised when DPD values cannot be assigned to valid states.

Example Error Handling

from collection_fc import CollectionFCError, DataValidationError try: result = collection_fc.forecast(data) except DataValidationError as e: print(f"Data validation failed: {e}") except CollectionFCError as e: print(f"CollectionFC error: {e}")
